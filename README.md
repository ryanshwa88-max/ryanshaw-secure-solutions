---

## 📋 Overview

This static landing page showcases the company's services—CCTV, biometric access control, structured cabling, PBX systems, WiFi networking, and server infrastructure. It is built to convert visitors into leads through a clear call-to-action and an integrated contact form.

The page features:
- A tech-inspired, dark-themed UI with gold accents.
- A "Monitoring Wall" mockup in the hero section.
- Scroll-triggered reveal animations.
- A mobile-first, responsive layout.

---

## 🚀 Features

- **Service Showcase**: Highlights six core installation services with icons and imagery.
- **Pricing Tiers**: Three clear plans (Install Only, Monitored, Monitored + Secured).
- **Team Introduction**: Displays company directors and lead technicians.
- **Testimonial**: Social proof from a client (Allan Ngoya, CEO of TLS).
- **Contact Form**: Integrated with **Netlify Forms** for automatic lead capture.
- **Interactive Map**: Embedded Google Map showing the Nairobi location.
- **System Status Bar**: A subtle status indicator to build trust.

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure.
- **CSS3** — Custom properties (variables), Flexbox, Grid, and animations.
- **Vanilla JavaScript** — Mobile menu toggle and Intersection Observer for scroll animations.
- **Google Fonts** — Space Grotesk, IBM Plex Sans, and IBM Plex Mono.
- **Netlify Forms** — Serverless form handling (no backend required).

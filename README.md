## 📌 Overview

This repository contains the complete source code for the **RyanShaw Secure Solutions** corporate website. It is a single‑page, mobile‑first, static HTML site built to showcase the company’s services, monitoring plans, team, and client testimonials — all while maintaining a sleek, security‑inspired dark theme.

The page is fully functional with:
- A sticky navigation bar and mobile hamburger menu.
- An interactive “monitoring wall” visual in the hero section.
- A working contact form (ready for Netlify or any backend form processor).
- Smooth scroll‑triggered animations.
- Embedded Google Map for location context.

---

## ✨ Features

- ✅ **Responsive & Accessible** — Works flawlessly on desktops, tablets, and phones. Built with semantic HTML and `:focus-visible` for keyboard navigation.
- ✅ **Modern Dark UI** — Clean typography with `Space Grotesk` and `IBM Plex Sans`, plus a subtle amber accent colour (#E8A33D).
- ✅ **Sticky Status Bar** — Shows a live “System Online” indicator with animated pulse.
- ✅ **Service Showcase** — Six core services (CCTV, Access Control, Cabling, PBX, WiFi, Servers) with custom SVG icons.
- ✅ **Tiered Monitoring Plans** — Three packages (Install Only / Monitored / Monitored+Secured), including a “Most Requested” featured plan.
- ✅ **Team Section** — Displays key staff with avatars and role descriptions.
- ✅ **Testimonial Block** — Social proof from a satisfied client.
- ✅ **Contact Form** — Built‑in support for Netlify Forms (with honeypot spam protection) and a map embed.
- ✅ **Scroll Animations** — Elements fade in as they enter the viewport (respects `prefers-reduced-motion`).

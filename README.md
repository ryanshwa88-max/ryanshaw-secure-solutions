## 🏢 About This Project

RyanShaw Secure Solutions is a professional website for a security installation company based in Nairobi, Kenya. The site showcases services including CCTV installation, biometric access control, structured cabling, PBX systems, WiFi networking, and server infrastructure.

## 🚀 Live Demo

[View the live site](https://your-site.netlify.app)

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Space Grotesk, IBM Plex Sans, IBM Plex Mono)
- Netlify Forms (for contact form)
- Google Maps Embed API

## 📱 Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Interactive monitoring wall display
- ✅ Service showcase with animated icons
- ✅ Pricing/Monitoring plans
- ✅ Team member profiles
- ✅ Client testimonials
- ✅ Contact form with Netlify integration
- ✅ Google Map showing Nairobi location
- ✅ Scroll reveal animations
- ✅ Mobile hamburger menu
- ✅ Status bar with live system indicator
- ✅ Accessibility features (focus states, reduced motion)
